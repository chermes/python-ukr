# python-ukr
Unsupervised Kernel Regression (UKR) for Python/Numpy

## Requirements
* Python 2.7
* SciPy
* Numpy
* scikit-learn
* Numexpr [optional]
* Matplotlib [for tests]

Note for Windows users: All packages are available in the WinPython distribution.

## Example
See src\_naive/test.py.
