# python-ukr
Unsupervised Kernel Regression for Python/Numpy

## Requirements
* Python 2.7
* SciPy
* Numpy
* scikit-learn
* Numexpr [optional]
* Matplotlib [for tests]

Additional requirements for the accelerated UKR:
* Cython

Note for Windows users: All packages are available in the WinPython distribution.
